### Check Operator Geographic Filter
```bash
# Login as operator
TOKEN=$(curl -s -X POST http://localhost:8000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"operator1@ziamis.gov.zm","password":"Operator1@2024"}' \
  | jq -r .access_token)

# List farmers (should only see Kawambwa District)
curl -s -H "Authorization: Bearer $TOKEN" \
  "http://localhost:8000/api/farmers?limit=100&skip=0" \
  | jq -r '.[].district_name' | sort | uniq -c
```

## üîí Security Notes

1. **Password Hashing:** All passwords use bcrypt via passlib
2. **JWT Tokens:** 30-minute access tokens, 7-day refresh tokens
3. **NRC Privacy:** NRCs are hashed (HMAC-SHA256) in the database
4. **Farmer IDs:** Generated from NRC hashes (deterministic)
5. **Geographic Filtering:** Operators cannot see farmers outside their districts
